Splitter (Unity3D)

Splitter is simple 2D mobile game available for Android made using Unity3D engine.

Citadel (Android Studio)

Citadel is an Android app for efficient and smooth sharing of books in college campus, promoting a healthy reading culture.

  • Contributed on android app that interacts with the backend(written using django) using Retrofit API.
  • Worked on various APIs and libraries to make our tasks easier.
  • Secured 1st prize in Mobile development competition of Shristi(Techno-Hobby Exhibition, IIT Roorkee)

Skip-The-Queue (Android Studio)

Skip-The-Queue is an Android app which aims to reduce queues(in Atms, popcorn vendors during movie intervals, etc.).

  • Worked on the android app, Microsoft Azure Services and the integration of the two
  • Worked on using various APIs(msg91 api for sending OTP, etc.) and libraries.
  • Secured 3rd prize in ONCAMPUS round and in Top 10 teams in SHOWCASE round in Microsoft Code.fun.do hackathon.

Hitball (libGdx)

Hitball is a 2D mobile game available for Android made using libGdx framework, where you have to keep the ball from falling by hitting it continuously.

  • Worked on themes for the game(day and night)
